## 1. Headings

```md
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## 2. Text Styling

```md
*Italic* or _Italic_
**Bold** or __Bold__
~~Strikethrough~~
```

## 3. Lists

**Unordered List:**

```md
- Item 1
- Item 2
  - Subitem
```

**Ordered List:**

```md
1. First
2. Second
   1. Sub-step
```

## 4. Links

```md
[Piyush](https://github.com/piyerx)
```

## 5. Images

```md
![Alt Text](https://example.com/image.png)
```

## 6. Code

**Inline code:**

```md
Use `code` for inline code.
```

**Code block:**

<pre>
```language
function hello() {
  console.log("Hello World");
}
```
</pre>

## 7. Blockquotes

```md
> This is a blockquote.
>> Nested blockquote.
```

## 8. Tables

```md
| Name  | Age |
|-------|-----|
| ABC  | 20  |
| PQR | 22  |
```

## 9. Task Lists

```md
- [x] Completed Task
- [ ] Incomplete Task
```

## 10. Horizontal Rule

```md
---
```

## 11. Escaping Characters

Use backslash (`\`) to escape characters:

```md
\*Not italic\*
```
